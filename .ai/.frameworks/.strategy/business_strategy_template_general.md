# Business Strategy Template

### General-Purpose Framework | Built on McKinsey, Porter, Gartner, BCG & Bain Research

> **How to use this template:**
> Work through each section in order. Do not skip sections — each one feeds the next.
> A strategy that skips analysis and jumps to goals is just a wish list.
> Recommended review cycle: **Quarterly** for KPIs, **Annually** for the full strategy.

---

## PHASE 1 — UNDERSTAND BEFORE YOU DECIDE

---

## 1. Executive Summary

**What to write:**
Write this section **last**. Summarize the entire strategy in 4–6 sentences. Cover: what the business does, who it serves, what makes it different, and what the #1 strategic priority is for the coming year.

> This section should be clear enough that a new stakeholder can read it in 60 seconds and understand the entire direction of the business.

**Why it matters:**
McKinsey research shows that alignment across teams is a prerequisite for execution. Without a shared summary, departments pursue different interpretations of the same strategy.
_(Source: McKinsey — Strategy and Corporate Finance Special Collection)_

---

## 2. Vision & Mission

### Vision _(3–5 Year Horizon)_

**What to write:**
One sentence. Where do you want this organization to be in 3–5 years? Focus on position and impact, not financial targets. Make it ambitious but grounded.

> Good vision statements answer: _What kind of organization do we want to become?_

### Mission _(Present Day)_

**What to write:**
One to two sentences. What does the organization do today, for whom, and how? This should be operational and specific, not vague.

> Good mission statements answer: _What do we do, for whom, and why?_

### Core Values

**What to write:**
List 3–5 non-negotiable principles that guide decision-making. These are behavioral anchors, not marketing slogans. Each value should be able to fail a real decision — if a value never creates a difficult choice, it is not a real value.

**Why it matters:**
The McKinsey 7-S Framework places Shared Values at the **center** of all seven organizational elements. It identifies values as the anchor that connects strategy, structure, systems, staff, style, and skills.
_(Source: McKinsey 7-S Framework — documented by Peters & Waterman, widely applied in McKinsey consulting practice)_

## 3. External Environment Analysis — PESTLE

**What to write:**
For each factor, identify: _What is happening? How does it affect our business? Is it a threat or an opportunity?_

> PESTLE is best used as a **starting point** for strategy, not a conclusion. It surfaces the context your strategy must operate within.
> _(Source: Platform Executive — "Choosing the Right Framework", 2026)_

| Factor            | What is Happening                                           | Impact on Business | Threat / Opportunity |
| ----------------- | ----------------------------------------------------------- | ------------------ | -------------------- |
| **Political**     | Government policies, trade regulations, political stability |                    |                      |
| **Economic**      | Inflation, interest rates, GDP growth, consumer spending    |                    |                      |
| **Social**        | Demographics, cultural shifts, consumer behavior trends     |                    |                      |
| **Technological** | Emerging technologies, automation, digital transformation   |                    |                      |
| **Legal**         | Industry regulations, labor law, compliance requirements    |                    |                      |
| **Environmental** | Sustainability trends, climate policy, ecological risks     |                    |                      |

**How to use the output:**
The most significant factors from this table become inputs to your SWOT analysis (Section 4) and your Risk Register (Section 10).
_(Source: Flevy / BCG & Bain integrated frameworks; SafetyCulture PESTLE guidelines)_

## 4. Industry Analysis — Porter's Five Forces

**What to write:**
Rate each force as Low / Medium / High and explain why. The overall picture tells you how attractive the industry is and where your strategic vulnerabilities lie.

> Porter's Five Forces was introduced by Harvard Business School professor Michael E. Porter in his 1979 Harvard Business Review article _"How Competitive Forces Shape Strategy"_ and expanded in his 1980 book _Competitive Strategy_. It remains one of the most widely used tools for competitive landscape analysis.
> _(Source: Porter, M., Harvard Business Review, 1979; Canva Strategic Guide, 2025)_

| Force                             | Intensity (L/M/H) | Key Factors                                                 | Strategic Implication |
| --------------------------------- | ----------------- | ----------------------------------------------------------- | --------------------- |
| **Competitive Rivalry**           |                   | Who are the main rivals? How aggressive are they?           |                       |
| **Threat of New Entrants**        |                   | How easy is it to enter this market? What are the barriers? |                       |
| **Bargaining Power of Buyers**    |                   | Can customers easily switch? Are they price-sensitive?      |                       |
| **Bargaining Power of Suppliers** |                   | How many suppliers exist? Can they dictate terms?           |                       |
| **Threat of Substitutes**         |                   | What alternatives exist for the customer?                   |                       |

**Overall Industry Attractiveness:** Low / Medium / High

> 💡 Combine Five Forces with PESTLE for a complete external picture. Five Forces tells you about competitive dynamics; PESTLE tells you about the macro environment shaping them.
> _(Source: Flevy — "Integrating PESTLE with Porter's Five Forces", Bain-endorsed approach)_

## 5. Internal Analysis — SWOT

**What to write:**
Be honest and specific. Avoid generic statements. Each point should be evidenced by something observable — a metric, a customer behavior, a capability, or a resource.

> ⚠️ Important: SWOT is most powerful as a **synthesis tool**, not a starting point. Use Sections 3 and 4 before filling this in. Generic SWOTs (e.g., "strong team", "competitive market") are symptoms of skipping prior analysis.
> _(Source: Platform Executive — "Choosing the Right Framework", 2026)_

|              | **Helpful**                                                                                               | **Harmful**                                                                                |
| ------------ | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| **Internal** | **Strengths** — What do we do better than anyone else? What resources or capabilities are unique to us?   | **Weaknesses** — Where do we lose ground to competitors? What capabilities are we missing? |
| **External** | **Opportunities** — What trends or market gaps can we exploit? What do our strengths enable us to pursue? | **Threats** — What external forces could damage us? Which weaknesses make us vulnerable?   |

**TOWS Strategic Options** _(convert SWOT into action)_

|                | Opportunities                                    | Threats                                  |
| -------------- | ------------------------------------------------ | ---------------------------------------- |
| **Strengths**  | S-O: Use strengths to capture opportunities      | S-T: Use strengths to neutralize threats |
| **Weaknesses** | W-O: Address weaknesses to capture opportunities | W-T: Minimize weaknesses, avoid threats  |

_(Source: TOWS Matrix methodology; SWOT integration with PESTLE and Five Forces — BCG & Deloitte frameworks)_

## 6. Strategic Positioning & Value Proposition

### 6.1 Strategic Position

**What to write:**
Choose where you compete and how. Based on Porter's generic strategies, pick **one** primary position and justify it:

| Position            | Description                                              | Choose if…                                   |
| ------------------- | -------------------------------------------------------- | -------------------------------------------- |
| **Cost Leadership** | Lowest cost producer in the industry                     | You have scale advantages or lean operations |
| **Differentiation** | Unique product or experience customers pay a premium for | You have a distinctive capability or brand   |
| **Focus (Niche)**   | Serve a specific segment better than broad competitors   | You have deep expertise in a narrow market   |

> Porter's research shows that being "stuck in the middle" — trying to compete on both cost and differentiation without committing — typically leads to below-average performance.
> _(Source: Porter, M. — Competitive Strategy, Free Press, 1980)_

### 6.2 Value Proposition

**What to write:**
One clear statement that answers: _Why should your target customer choose you over every alternative?_

> **Formula:** _"We help [target customer] to [solve a specific problem / achieve a specific outcome] by [your unique approach], unlike [the alternative] which [its limitation]."_

A strong value proposition is: specific (not "best quality"), tied to a real customer pain, and defensible over time.

---

## 7. Target Market & Customer Segmentation

### 7.1 Primary Segment

**What to write:**
Describe your ideal customer with enough specificity that it could guide a hiring decision or a marketing brief. Include: demographics, psychographics, key behaviors, and the core job they are trying to get done.

> Use the **Jobs-To-Be-Done (JTBD)** lens: customers don't buy products, they hire them to do a job. Define the job your customer is hiring your product/service for.
> _(Source: Strategypunk.com — Three Horizons & JTBD integration, 2026)_

### 7.2 Secondary Segment _(if applicable)_

**What to write:**
A smaller segment you also serve. Define how their needs differ from the primary segment and whether you serve them with the same or a different offer.

### 7.3 Customer Insight Summary

**What to write:**
What do you know about your customer that your competitors don't? What unmet need, frustration, or aspiration drives their decision?

## 8. Growth Strategy — McKinsey Three Horizons

**What to write:**
Classify your current and planned initiatives into three horizons. This prevents the common failure mode where short-term pressure consumes all resources and starves future growth.

> The Three Horizons model was developed by McKinsey & Company and published in _The Alchemy of Growth_ (Baghai, Coley, White). McKinsey research shows companies with a balanced portfolio across all three horizons deliver a **30% greater return on investment** than those focused solely on short-term results.
> _(Source: McKinsey — "Enduring Ideas: The Three Horizons of Growth"; Flevy McKinsey Three Horizons case study)_

| Horizon                         | Focus                                                               | Time Frame      | Your Initiatives              |
| ------------------------------- | ------------------------------------------------------------------- | --------------- | ----------------------------- |
| **H1 — Defend & Extend Core**   | Optimize existing business, protect revenue, improve margins        | Now → 12 months | List your H1 initiatives here |
| **H2 — Build Emerging Engines** | Scale new products, markets, or models that show early traction     | 1–3 years       | List your H2 initiatives here |
| **H3 — Create Future Options**  | Explore and experiment with ideas that could become tomorrow's core | 3–5+ years      | List your H3 initiatives here |

**Resource Allocation Guidance:**
A commonly referenced starting point is the **70 / 20 / 10 rule** (H1 / H2 / H3), though this should be adapted to your context and growth stage. The key principle is that H2 and H3 must receive _explicit, protected_ investment — otherwise H1's operational pressure will consume everything.
_(Source: McKinsey Three Horizons framework; Umbrex strategic resource allocation guidance, 2026)_

## 9. Strategic Objectives & OKRs

**What to write:**
Translate the strategy into a small number of measurable commitments. Use the OKR (Objectives and Key Results) format.

> OKRs were developed at Intel by Andy Grove in the 1970s and popularized by John Doerr at Google in 1999. Google's adoption of OKRs is widely cited as a cornerstone of its management success.
> _(Source: Scopy.me — "How the OKR Framework Aligns with McKinsey Three Horizons", 2025)_

> **OKR Structure:**
>
> - **Objective:** An ambitious, qualitative goal. It should be inspiring and clear. _"Build the most trusted brand in [category]."_
> - **Key Results (2–4 per Objective):** Specific, measurable outcomes that prove the objective is being achieved. Not tasks. _"Achieve NPS score of 60+ by Q4."_

## 10. Risk Management

**What to write:**
Identify the 4–6 biggest risks to the strategy. For each: assess likelihood and impact, and define a concrete mitigation action.

> Strategy without risk assessment is overconfidence. McKinsey research identifies strategic risk identification and mitigation as one of the ten tests of a good strategy.
> _(Source: McKinsey — Strategy and Corporate Finance Special Collection; "Ten Tests of a Good Strategy")_

> Recommended risk categories to consider: market risk, competitive risk, operational risk, financial risk, regulatory risk, talent risk, technology risk.

## 11. Performance Measurement — Balanced Scorecard

**What to write:**
Define KPIs across four perspectives. This ensures you measure the **full health** of the organization, not just financial results.

> The Balanced Scorecard was developed by Robert Kaplan and David Norton at Harvard Business School. Accenture research indicates that leading companies use a balanced scorecard approach — combining leading and lagging indicators — to measure performance across short and long-term horizons.
> _(Source: Flevy McKinsey Three Horizons case study citing Accenture; Bob Stanke — "Strategic Planning Models", 2025)_

### Financial Perspective

_Are we creating value for our stakeholders?_

| KPI                 | Definition                     | Baseline | Target | Frequency |
| ------------------- | ------------------------------ | -------- | ------ | --------- |
| Revenue Growth      | YoY revenue increase %         |          |        | Quarterly |
| Gross Margin %      | Gross Profit ÷ Revenue         |          |        | Monthly   |
| Operating Cash Flow | Cash generated from operations |          |        | Monthly   |

### Customer Perspective

_Are we delivering value to our customers?_

| KPI                          | Definition                                 | Baseline | Target | Frequency |
| ---------------------------- | ------------------------------------------ | -------- | ------ | --------- |
| Net Promoter Score (NPS)     | Customer loyalty score (-100 to +100)      |          |        | Quarterly |
| Customer Retention Rate      | % of customers retained period over period |          |        | Monthly   |
| Customer Satisfaction (CSAT) | Average satisfaction score                 |          |        | Monthly   |

### Internal Process Perspective

_Are our operations efficient and effective?_

| KPI                    | Definition                          | Baseline | Target | Frequency |
| ---------------------- | ----------------------------------- | -------- | ------ | --------- |
| [Core process metric]  | Define based on your business model |          |        |           |
| [Quality / error rate] | Define based on your business model |          |        |           |
| [Cycle time / speed]   | Define based on your business model |          |        |           |

### Learning & Growth Perspective

_Are we building the capabilities for the future?_

| KPI                       | Definition                                           | Baseline | Target | Frequency   |
| ------------------------- | ---------------------------------------------------- | -------- | ------ | ----------- |
| Employee Engagement Score | Internal survey score                                |          |        | Semi-annual |
| [Innovation metric]       | % revenue from new products/services                 |          |        | Annual      |
| [Capability development]  | Training hours, certifications, or skill gaps closed |          |        | Quarterly   |
